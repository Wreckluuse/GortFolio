This is a WIP portfolio application by Andrew Weightman. It leverages Golang and HTMX to make calls to various public quote and advice API's and renders the response in your browser. Check back over time and see what fun features get added as I expand my knowlege!

Planned Features

  • Authentication to allow users to sign in.
  • Database integration to make quotes savable.
  • Make quotes shareable on various social media platforms.
